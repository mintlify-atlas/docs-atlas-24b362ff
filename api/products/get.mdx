---
title: 'Get Product'
api: 'GET /products/:id'
description: 'Retrieve a single product by its ID'
---

This endpoint returns the complete information for a specific product identified by its UUID.

## Path Parameters

<ParamField path="id" type="string" required>
  Product UUID identifier
  
  **Validation:** Must be a valid UUID format
</ParamField>

## Response

<ResponseField name="error" type="boolean">
  Indicates if an error occurred during the request
</ResponseField>

<ResponseField name="msg" type="string">
  Response message ("Product got sucessfully" on success)
</ResponseField>

<ResponseField name="product" type="object">
  The product object containing all product details
  
  <expandable title="product properties">
    <ResponseField name="id" type="string">
      Product UUID identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Product name
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Product description
    </ResponseField>
    
    <ResponseField name="price" type="string">
      Product price in decimal format (e.g., "3000.00")
    </ResponseField>
    
    <ResponseField name="stock" type="integer">
      Available stock quantity
    </ResponseField>
    
    <ResponseField name="category_id" type="integer">
      ID of the category this product belongs to
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Timestamp when the product was created (ISO 8601 format)
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      Timestamp when the product was last updated (ISO 8601 format)
    </ResponseField>
  </expandable>
</ResponseField>

## Success Response Example (200)

```json
{
  "error": false,
  "msg": "Product got sucessfully",
  "product": {
    "id": "b6da7bbc-14b9-11f1-9fcd-2418c6c96a00",
    "name": "Coca cola",
    "description": "Bebida gasificada de 2ltr",
    "price": "3000.00",
    "stock": 40,
    "category_id": 1,
    "created_at": "2026-02-28T15:25:29.000Z",
    "updated_at": "2026-02-28T15:25:29.000Z"
  }
}
```

## Error Responses

### 400 - Invalid UUID

```json
{
  "error": true,
  "msg": "the id is invalid"
}
```

### 409 - Conflict

Returned when the product is not found or a database error occurs.

```json
{
  "error": true,
  "msg": "Error message describing the issue"
}
```
