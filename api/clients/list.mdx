---
title: "List Clients"
api: "GET /clients"
description: "Retrieve all clients in the system"
---

This endpoint returns a list of all clients registered in the system.

## Response

<ResponseField name="error" type="boolean">
  Indicates if there was an error (always false on success)
</ResponseField>

<ResponseField name="listClients" type="array">
  Array of client objects
  
  <Expandable title="Client Object">
    <ResponseField name="id" type="string">
      Client's unique identifier (UUID format)
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Client's full name
    </ResponseField>
    
    <ResponseField name="email" type="string">
      Client's email address
    </ResponseField>
  </Expandable>
</ResponseField>
