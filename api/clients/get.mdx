---
title: "Get Client"
api: "GET /clients/:id"
description: "Retrieve a specific client by ID"
---

This endpoint retrieves detailed information about a single client using their unique ID.

## Path Parameters

<ParamField path="id" type="string" required>
  Client's unique identifier in UUID format
</ParamField>

## Response

<ResponseField name="error" type="boolean">
  Indicates if there was an error
</ResponseField>

<ResponseField name="client" type="object">
  Client information
  
  <Expandable title="Client Object">
    <ResponseField name="id" type="string">
      Client's unique identifier (UUID format)
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Client's full name
    </ResponseField>
    
    <ResponseField name="email" type="string">
      Client's email address
    </ResponseField>
  </Expandable>
</ResponseField>

## Response Example

<ResponseExample>
```json 200 - Success
{
  "error": false,
  "client": {
    "id": "49a5ddf8-14ce-11f1-9fcd-2418c6c96a00",
    "name": "albert Epstein",
    "email": "albertE@ralatividad.com"
  }
}
```

```json 400 - Invalid ID
{
  "error": true,
  "msg": "Id is invalid"
}
```

```json 409 - Client Not Found
{
  "error": true,
  "msg": "Client not found"
}
```
</ResponseExample>
